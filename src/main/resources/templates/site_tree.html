Помощь

Все посетители сайта имеют доступ к главной странице сайта, на которой представлен список всех опубликованных объявлений. Для перехода на главную страницу необходимо нажать левой кнопкой мыши (далее ЛКМ) на логотип сайта в левом верхнем углу экрана.
Для поиска необходимо ввести поисковый запрос в панель сыерху справа и нажать клавишу «Enter» или ЛКМ по кнопке с увеличительным стеклом.
Для просмотра подробной информации о товаре необходимо нажать ЛКМ по карточке товара на главной странице. В странице подробного просмотра товара можно оформить заказ на товар по нажатию кнопки «Оформить заказ», после которого появится окно с формой для заказа. После заполнения всех полей формы необходимо нажать кнопку «отправить». Заказ будет сразу перенаправлен к продавцу.
Для доступа к функционалу продавца, пользователю необходимо авторизоваться в системе. Сделать это можно по ссылке «Войти» справа сверху.
Для управления объявлениями перейдите по ссылке «Мои объявления». Каждое объявление представлено в виде карточки товара. Для удаления объявления нажмите на кнопку с изображением корзины (при наведении становится красного цвета) и подтвердите удаление.
Для редактирования товара нажните на кнопку с изображением карандаша (при наведении становится зеленого цвета). Вам станет доступна форма для редатирования объявления. После внесения изменений нажмите кнопку «Сохранить».
Добавление объявления: нажмите ЛКМ по пункту меню «Добавить объявление» или на карточку с подписью «Добавить объявление» на странице «Мои объявления».
Для просмотра информации о текущем авторизованном пользователе нажмите на иконку с пользователем справа сверху. Для выхода из системы нажмите на иконку с пользователем справа сверху и в выпавшем меню пункт «Выход».
